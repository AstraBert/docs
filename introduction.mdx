## What is PdfItDown?

**PdfItDown** is a python package that relies on [`markitdown` by Microsoft](https://github.com/microsoft/markitdown/), [`markdown_pdf`](https://github.com/vb64/markdown-pdf) and [`img2pdf`](https://pypi.org/project/img2pdf/) to carry on conversion of text-based files and images to PDF. **PdfItDown** is applicable to the following file formats:

- Markdown
- PowerPoint
- Word
- Excel
- HTML
- Text-based formats (CSV, XML, JSON)
- ZIP files (iterates over contents)
- Image files (PNG, JPG)


## Setting up

To set uo PdfItDown, it is good practice to create an isolated development environment

<CodeGroup>

  ```bash venv
  python3 -m venv .venv
  source .venv/bin/activate
  pip install pdfitdown
  ```

```bash conda
# you can use all python versions above 3.9
conda create -n pdfitdown python=3.11
conda activate pdfitdown
pip install pdfitdown
```

</CodeGroup>
