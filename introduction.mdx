## Setting up

To set uo PdfItDown, it is good practice to create an isolated development environment

<CodeGroup>

  ```bash venv
  python3 -m venv .venv
  source .venv/bin/activate
  pip install pdfitdown
  ```

```bash conda
# you can use all python versions above 3.9
conda create -n pdfitdown python=3.11
conda activate pdfitdown
pip install pdfitdown
```

</CodeGroup>
